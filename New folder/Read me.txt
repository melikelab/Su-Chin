The program 'callVoronoiMultiSeg' is for Voronoi tesselation and cluster finding
The program 'PostVA_mod' is for post analysis of all the data files obtained after Voronoi tesselation and 
cluster analysis. 
The program 'CumulativeProbDist_VA_NORMALIZED' provides the cumulative probability densities and acepts the '.mat' output 
from the 'PostVA_mod'

The folder 'Accesory files' contain all the relevant functions to run these 2 main programs.
